Features

Everything you need to work with AI coding agents, in one app.

🤖

Agent Agnostic

Works with any ACP-compatible agent — Claude Code, Gemini CLI, Codex CLI, GitHub Copilot, Junie, Kimi, and more. You're never locked in.

🔎

ACP Registry

Browse, search, and install agents from the central ACP agent registry. One click to add a new agent to your workflow.

🌳

Git Worktree Isolation

Optionally isolate each thread on its own branch with git worktrees. Parallel tasks never interfere with each other.

⚡

Streaming Responses

See text, thinking blocks, and tool calls in real time as the agent works. No waiting for the full response.

📑

Diff Review & Commit

See exactly what files the agent changed with inline diffs. Stage and commit changes right from the app.

🔒

Permission System

Approve or deny agent actions like file writes and command execution via UI dialogs. You stay in control.

💻

Built-in Terminal

Per-session terminal for testing agent changes immediately. No need to switch windows.

🛠

Multi-Workspace Support

Organize your work across multiple workspaces. Run different agents on different projects simultaneously.
